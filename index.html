Hyperspectral Images (HSI) processing (spectral unmixing and classification)

This project focuses on the analysis of hyperspectral imaging data, specifically the Salinas dataset, which is commonly used in remote sensing applications. The dataset contains high-dimensional spectral data of agricultural fields, captured across multiple wavelengths. The primary objectives of this project include data preprocessing, feature extraction, and classification using various machine learning techniques.

This project successfully demonstrates the application of machine learning techniques to hyperspectral image classification. By leveraging spectral data and endmember information, it achieves the classification of agricultural fields into various crop types with a reasonable degree of accuracy. The project highlights the importance of preprocessing and feature extraction in handling high-dimensional data typical in remote sensing.

Athens AirBnB Statistical Analysis

This project is centered around analyzing Airbnb listings in Athens using data obtained from the Inside Airbnb platform. The objective is to explore and understand various aspects of the Airbnb market in Athens, such as neighborhood popularity, pricing trends, availability, and income potential for hosts. The analysis aims to provide insights into how Airbnb impacts the local housing market and tourism industry.

Handwritten Digits Classification with SVD

The project focuses on classifying handwritten digits using data preprocessing, Singular Value Decomposition (SVD), and subsequent classification techniques. The primary goal is to achieve accurate digit recognition by leveraging SVD to decompose and analyze the data before applying classification algorithms.

This project showcases the application of advanced linear algebra techniques like SVD in practical machine learning tasks, particularly in image recognition. The successful implementation of SVD and its use in enhancing classification accuracy are the significant achievements of this work.

Political Campaigns Statistical Analysis

The analysis is part of a larger study on political movements, focused on understanding the frequency, success rates, and other characteristics of these campaigns over time.

The project highlights the application of data science to study socio-political phenomena. The key achievements include successfully preprocessing and analyzing complex historical data, generating insightful visualizations, and applying statistical models to derive meaningful conclusions about the dynamics of political campaigns.

Text Analytics - POS Tagger (MLP,RNNs,CNNs)

The third project focuses on Natural Language Processing (NLP) using machine learning techniques to classify Part-of-Speech (POS) tags in Italian text. The project involves data preparation, word embedding generation using FastText, and training a Multi-Layer Perceptron (MLP), an RNN and a CNN model for classification.

FastText word embeddings are used to convert words into numerical vectors. The project specifically utilizes Italian word embeddings, where each word is represented as a 300-dimensional vector. For each window of three words, these vectors are concatenated to form a 900-dimensional vector, which serves as input to the model.
